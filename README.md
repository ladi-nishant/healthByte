## Features 
- Blockchain fundraising 
- Doctor/patient booking 
- Video consultations (WebRTC) 
- E-commerce for medicines 
 
## Installation 
1. Clone repo 
2. Install dependencies: npm install 
3. Start: npm run dev 
 
## Usage 
Run the app locally and visit http://localhost:3000
